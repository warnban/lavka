Фон баннера главной страницы
============================

Чтобы поставить фото на фон баннера — положите сюда файл:

    hero-church.jpg

Рекомендации:
- формат: JPG или WebP
- ширина: 1600–1920 px
- сюжет лучше смещать вправо (слева баннер затемняется под заголовок)

Пока файла нет — баннер показывает фирменный градиент (заглушка),
заголовок и кнопка остаются на месте.

На боевом сайте (Битрикс) этот источник картинки заменяется
на поле «изображение» из инфоблока «Баннеры», и фон меняется
из админки без правки кода.
