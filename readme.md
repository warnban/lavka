# Рогожская Лавka — демо-макет

Статический HTML-прототип интернет-магазина для согласования дизайна и структуры страниц с заказчиком.

## Что это

- Демонстрация внешнего вида и навигации по сайту
- Не CMS: товары и заказы не работают по-настоящему
- Рабочая версия на 1С-Битрикс — отдельно, на следующем этапе

## Страницы

Главная, каталог, карточка товара, корзина, скидки, о нас, доставка, личный кабинет, поиск.

## GitHub Pages

После публикации репозитория включите **Settings → Pages → Deploy from branch → main / root**.

Сайт откроется по адресу: `https://warnban.github.io/lavka/`
